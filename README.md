# Piano App für meinen Enkel

Diese App ist ein interaktives Klavier, das mit **Flutter** entwickelt wurde. 
Ich erstelle sie als besonderes Geschenk für meinen zehnjährigen Enkel, 
um ihm Freude an der Musik und Technik zu vermitteln.

## Geplante Funktionen
* Interaktive Klaviertasten zum Spielen
* Verschiedene Klänge
* Einfache Bedienung für Kinder
